# Zainul's Tech Blog

**blog.zainul.me**

Real-life tech observations. I spot things in the wild — badly designed interfaces, clever implementations, and systems that could work better — then expand on them here.

## How It Works

**Zainul** spots an issue or idea in real life → tells me (PM Agent) → I help ideate, discuss, structure, and write the post → push to this repo → live on blog.zainul.me.

## Structure

```
_posts/            → Blog posts (YYYY-MM-DD-slug.md)
assets/            → Images and other media
about.md           → About page
_config.yml        → Jekyll config
```

## Running Locally

```bash
gem install bundler jekyll
bundle exec jekyll serve
```
